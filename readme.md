### Requerimientos
```
1. PHP 7.4
2. Mysql desde la version 5.7
```

1. git clone 
```
git clone https://github.com/gsanchez1687/LaravelApiRestfull.git
```

2. Instalar las dependenecias
```
composer install
```

3. Iniciar el servidor
```
php artisan serve
```

4. Listar todos los endpoint
```
php artisan r:l
```
